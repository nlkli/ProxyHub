Сервера

Вы можете выбрать любой сервер из списка, ориентируясь на его текущую нагрузку и другие параметры. По мере роста количества пользователей будут добавлятся новые сервера.

Каждый сервер поддерживает несколько протоколов подключения. Для каждого из них может быть доступно несколько адресов — они работают одинаково, но используют параметры подключения для лучшего распределения трафика. Можно выбрать любой из них случайным образом или все сразу.
