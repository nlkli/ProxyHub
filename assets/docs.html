Введение

ProxyHub предоставляет доступ к серверам, использующим различные сетевые протоколы для проксирования и безопасной передачи трафика. Выбор протокола определяет, как данные передаются, шифруются и маршрутизируются между клиентом и сервером.

На данный момент сервера ProxyHub поддерживают подключение по трём основным протоколам:

  • Xray VLESS Reality — современный протокол для безопасного и эффективного обхода блокировок, обеспечивающий шифрование и скрытие сетевого трафика;
  • HTTP Proxy — универсальный прокси для работы с веб-трафиком и совместимости с большинством приложений;
  • SOCKS5 Proxy — гибкий протокол для передачи любых типов данных с поддержкой аутентификации и минимальной задержкой.

Эти три протокола покрывают основные сценарии использования: от безопасного серфинга в интернете до подключения приложений, требующих прокси. Выбор подходящего протокола зависит от конкретных задач и предпочтений пользователя.

Данная документация не углубляется в детали внутренней работы протоколов или их взаимодействие на сетевом уровне. Её цель — объяснить, для каких задач следует использовать каждый протокол, а также предоставить пошаговые инструкции по настройке и подключению клиентов.

Xray-core

Xray-core — это мощный сетевой инструмент с открытым исходным кодом, который используется для проксирования трафика и обхода интернет-ограничений (интернет-цензуры). Xray-core поддерживает широкий спектр протоколов, что делает его универсальным инструментом для разных задач.

Изначально разработан для эффективного обхода систем интернет-цензуры в странах с жесткими ограничениями, таких как Россия, Иран и Китай.

VLESS Reality

ProxyHub использует VLESS Reality в качестве основного протокола из Xray-core.

Протокол VLESS остается незамеченным, потому что он маскирует весь свой трафик под обычный зашифрованный интернет-трафик (HTTPS), который используют все легальные веб-сайты. Таким образом его практически невозможно отследить и заблокировать.

REALITY — это дополнительная функция обфускации (маскировки) для протокола VLESS, которая выводит скрытность на новый уровень. Она позволяет серверу имитировать реальный, легитимный веб-сайт, до которого уже доверяют системы цензуры.

Ваш интернет-трафик полностью шифруется с помощью того же стандартного протокола TLS 1.3, который используют все безопасные веб-сайты по HTTPS.

Как подключить Xray VLESS Reality

Для подключения нужен установленный клиент с поддержкой ядра Xray-core и специальная ссылка подключения. Пример ссылки: 

vless://3cd59152-1b81-4999-ba65-14121d97ce8c@175.223.82.25:21229?security=reality&sni=github.com&pbk=HTCd5J980Uc7DAg3QEZ-oAvsI50e1kU2AwmxhEr62jM&sid=6518745g587975e8&type=tcp&flow=xtls-rprx-vision&encryption=none#user4(not valid just an example)

Эта ссылка содержит все параметры для подключения: IP адрес сервера, порт, тип шифрования и идентификатор пользователя.

Алгоритм подключения для всех клиентов (приложений) примерно одинаков. Вам нужно импортировать ссылку для подключения в клиент, после чего приложение автоматически настраивает все параметры, и вы можете подключиться к защищенному интернету.

Важно

  • Сохраняйте ссылки подключения в безопасном месте.
  • Используйте официальные клиенты для безопасности.

Клиенты для подключения

Существуют различные клиентские приложения для всех популярных операционных систем. В этом руководстве будут рассмотрены

Для мобильных устройств

  • iPhone / iOS: v2RayTun (App Store)

  • Android: v2RayTun (Google Play)

Краткая инструкция для мобильных устройств

1. Установите приложение v2RayTun.

2. Добавьте ссылку VLESS Reality через функцию "Импорт ссылки".

3. Включите подключение и проверьте соединение.
